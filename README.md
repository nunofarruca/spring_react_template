# spring react prototype

* Java 17
* Gradle 7
* Spring 2.7
* React 18
* Webpack 5
* Ts-Loader
* Eslint 8
* Prettier
* Lodash

## Instructions
To run in prod mode (http://localhost:8080):

```bash
./gradlew clean build bootRun
```

To run on dev mode:
Backend, runs on http://localhost:8080
```bash
./gradlew bootRun
```

Frontend, navigate to ./frontend, runs on http://localhost:4000:
```bash
npm start
```
